# 📖 Project Description

The Weather Forecast App is an Android application developed in Java using Android Studio. It provides real-time weather information using the WeatherAPI. Users can search weather by city name or use their current location to get live weather updates.

---

# ✨ Features

- 🌍 Search weather by city
- 📍 Current location weather
- 🌡 Current temperature
- ☁ Weather condition
- 💧 Humidity
- 🌬 Wind speed
- 🌧 Rain chance
- ☀ UV Index
- 🕒 Hourly Forecast
- 📅 7 Days Forecast
- 🖼 Dynamic weather icons
- 🎨 Modern UI Design

---

# 🛠 Technologies Used

- Java
- Android Studio
- XML
- Volley
- Glide
- RecyclerView
- WeatherAPI

# 🎯 Project Scope

This application helps users check live weather conditions of any city with an easy-to-use and attractive interface.
